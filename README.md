## プロダクト概要
### 経緯
サポーターズが主催する【技育CAMP】マンスリーハッカソン vol.2に参加した際に即席チームで作成した。チームの大半が24、25卒であったため、多くが就活に関して何かしらの悩みを抱えていた。また、当時はchatGPTのAPIが公開されたばかりであったため、chatGPTを使用したプロダクトを作成したいという声があがった。
以上から、面接対策が可能な就活LINEbotを作成することとなった。
